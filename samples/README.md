# Sample images

These are **fully synthetic, license-free (CC0)** placeholder images — gradients
and soft shapes generated by [`generate_samples.py`](./generate_samples.py), not
photographs. You can use, modify, and redistribute them freely.

Use them to try the app or regenerate your own:

```bash
python samples/generate_samples.py
```

For real collages, upload your own photos in the app — nothing here is required.
